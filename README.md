# Give me pixels
Recreate an image with a given color palette.\
How to use:

<span style="color:yellow">./give_me_pixels</span>
<span style="color:green">path_to_image_file</span>
<span style="color:orange">path_to_palette_file</span>

OR 

<span style="color:yellow">./give_me_pixels</span>
<span style="color:green">path_to_image_file</span>

When no palette file is given then the default color palette is used.

## Default application color palette
MC FUPERY ARTMAP SHADES (GG) PALETTE\
Created by Gengyen\
(GG stands for Gengyen)\
https://lospec.com/palette-list/mc-fupery-artmap-shades-gg

## Online palette list
https://lospec.com/palette-list

I use the PNG Image (1x) for downloading a file.
